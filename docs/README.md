# Docs directory index

This folder contains corpus source documents and supporting paths.

## Domain sources

- `docs/business/` for business domain documents.
- `docs/software-development/` for software-development domain documents.
- `docs/shared/` for cross-domain shared references.

## Special directories

- `docs/_drafts/` for in-progress documents excluded from exports.
- `docs/_deprecated/` for deprecated documents excluded from exports.
- `docs/_templates/` for reusable authoring templates excluded from exports.
